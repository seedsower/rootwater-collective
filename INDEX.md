## 🎯 COMMON TASKS

### Add Album Artwork
1. Drop image in `images/` folder
2. Edit `index.html` or `pages/music.html`
3. Replace placeholder `<div>` with `<img src="images/your-cover.jpg">`

### Change Accent Color
1. Open `css/main.css`
2. Line 13: Change `--gold: #D4A843;` to your hex value
3. Refresh browser

### Update Social Links
1. Search all files for `href="#"`
2. Replace with real URLs (e.g., `https://spotify.com/artist/...`)

### Connect Forms to Backend
1. Read [README.md](README.md#forms-integration)
2. Edit `js/main.js` → `initNewsletterForm()` and `initContactForm()`
3. Add your API endpoint

### Deploy to Netlify
1. Push to GitHub: `git push origin main`
2. Go to [netlify.com](https://netlify.com)
3. "New site from Git" → Select repo → Deploy
4. Done! Get live URL
